SISTEMA MOVIL PARA FACTURACIÓN

¿Eres distribuidor mayorista o minorista? Si tu punto de venta genera colas largas para tus clientes, puedes vender y generar comprobantes desde tu celular.

  • Buscador inteligente de productos, si cuentas con mas de 1000 productos, con solo una palabra podrás buscar y encontrar el producto que desees.
  • Si tienes una devolución de tus productos ¡Descuida! Podrás anular las transacciones y comprobantes desde el mismo dispositivo.
  • Observaciones adicionales, podrás agregar comentarios de tus clientes en tus productos para una atención optima.
  • Podrás vender desde cualquier parte, pero mejor aún, desde tu dispositivo móvil.
  • Generación y consulta de comprobantes electrónicos (Boletas, Facturas).
  • Especializado para venta itinerante.
  • Detalle de precio y stock actual de inventario.

SISTEMA MOVIL PARA RESTAURANTES

Con el sistema móvil tomar pedidos ya no será un problema. Las comandas móviles le permitirán generar ordenes de pedidos directamente a cocina/barra desde un celular o Tablet.

  • Panel de lista de pedidos, donde podrás observar todos los pedidos realizados antes de generar la orden en cocina/barra.
  • Cada mozo tendrá un usuario para la atención, por lo que podrás controlar la cantidad de mesas atendidas por mozo.
  • Controla el estado de los pedidos, veras si están pendientes de atención, atendidos o cancelados.
  • El panel de mesas te permitirá diferenciar que mesas están ocupadas y que mesas están libres.
  • Agrega observaciones a tus pedidos. ¡Algunos clientes no quieren el ceviche con picante!
  • El sistema móvil cuenta con una carta virtual interactiva para la toma de pedidos.
  • Podrás ver el precio de tus platos y modificar la cantidad.
